# Finance monitoring backend by c0de4fun

## Tech stack used

* Java 17
* Spring Boot
* Spring Security (JWT Based Authentication and Authorization)
* Spring Data JPA/Hibernate
* MySQL
* Lombok
* JSP
* Figma

### Setup

Install Java 17
Install Gradle
Install Docker

Run docker command

```
docker-compose up -d
```

Build Jar Fle

```
gradle build
```

Run Spring App

```
java -jar spring-app.jar
```
